Generated: 2025-09-03T19:49:53.868510 UTC

O que esta versão faz:
- Visual estilo "Gemini" (cards, status pill, strong typografia).
- Paciente já vê prescrição, orçamentos e exame com instruções e botão de confirmar.
- Médico tem painel simples para visualizar pacientes e criar prescrições (simulado).
- Demo 100% client-side (localStorage). Não envia dados. Use apenas para demonstração.

Como rodar localmente:
1. Extraia o ZIP.
2. Abra index.html no navegador.

Deploy no Vercel:
- Faça upload do ZIP no Vercel (Add New → Project → Import → Upload ZIP), ou
- Crie repositório no GitHub com os arquivos e importe no Vercel.

Arquivos:
- index.html, style.css, app.js, README.txt
